File accesso random
Come Funziona il Sistema di Accesso Random:

Caricamento dei dati:

Quando avvii il programma, il sistema inizia caricando tutti i dati degli studenti presenti nel file "ALUNNI.DAT". Questo avviene attraverso il metodo carica_vettore(), che legge i dati dal file e li memorizza nei vettori della memoria.

Navigazione nei dati:

Quando l'utente interagisce con l'interfaccia grafica, può richiedere operazioni come la visualizzazione, la modifica o l'eliminazione di record di studenti. Qui è dove entra in gioco l'accesso random. Utilizziamo il puntatore del file per posizionarci esattamente dove vogliamo all'interno del file, senza dover leggere ogni singolo record dall'inizio.

Operazioni sui dati:

Una volta posizionati nel punto desiderato del file, possiamo eseguire le operazioni richieste, come la lettura dei dati degli studenti per visualizzarli, la modifica dei record esistenti o l'inserimento di nuovi dati.

Chiusura del file:

È importante gestire correttamente il file dopo aver eseguito le operazioni. Assicuriamo sempre di chiudere il file in modo sicuro per salvare le modifiche e garantire l'integrità dei dati.
